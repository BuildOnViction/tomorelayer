# First


<a id="org4a69118"></a>

## Pre-requisite

-   Python 3.7 (recommended installing with pyenv)
-   pipenv
-   nvm
-   yarn
-   Docker


<a id="org9c19102"></a>

## Setup

-   Install required dependencies

        $ yarn
-   Fireup database with Docker

        $ yarn db
-   Install backend dependencies

        $ pipenv install
-   Start the app

        $ yarn backend

(&#x2026;frontend setup underdevelopment)
